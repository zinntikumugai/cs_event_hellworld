# Dart - Hello World

## Dart言語とは

Dart は2011年にGoogleが開発したプログラミング言語。Flutterフレームワークの開発言語として広く知られ、Web・モバイル・デスクトップ・サーバーサイドのクロスプラットフォーム開発を単一言語で実現する。AOT（事前コンパイル）とJIT（実行時コンパイル）の両方をサポートし、開発時の高速リロードと本番環境でのネイティブ性能を両立する。

## 特徴

- **健全な型システム**: null安全を含む静的型付けで、コンパイル時にバグを検出
- **非同期プログラミング**: async/await と Stream による直感的な非同期処理。イベント駆動アーキテクチャの基盤
- **Stream**: 非同期イベントの連続を表現するオブジェクト。`Stream.periodic` で一定間隔のイベント生成、`await for` で逐次消費
- **シングルスレッド + イベントループ**: JavaScript と同様のイベントループモデルで、非同期処理をシンプルに記述
- **AOTコンパイル**: ネイティブバイナリに事前コンパイルし、起動が高速でランタイム不要

## ビルドと実行

```bash
# Dockerイメージのビルド
docker build -t hello-dart .

# 実行
docker run --rm hello-dart
```
