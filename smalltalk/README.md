# Smalltalk - Hello World

## Smalltalk言語とは

Smalltalk は1970年代にXerox PARCで開発された純粋オブジェクト指向プログラミング言語。すべてがオブジェクトであり、メッセージパッシングによってオブジェクト間の通信を行う。GUIやIDEの概念を生み出した言語としても知られる。

## 特徴

- **純粋オブジェクト指向**: 数値やブール値を含め、すべてがオブジェクト
- **メッセージパッシング**: オブジェクトにメッセージを送ることで処理を実行
- **動的型付け**: 実行時に型が決定される
- **ライブ環境**: イメージベースの開発環境で、実行中にコードを変更可能
- **リフレクション**: プログラム自身の構造を実行時に調査・変更可能
- **シンプルな構文**: 予約語が極めて少なく、構文規則が単純

```bash

# 実行

```
